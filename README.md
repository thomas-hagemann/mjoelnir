This is going to be a small javascript library for **mulit selection**, **sorting** and **drag and drop**.

### Initial setup
```sh
# Clone the git repo
$ git clone https://github.com/thomas-hagemann/mjoelnir.git
$ cd mjoelnir

# Install all dependencies
$ npm install

# run
$ gulp watch
```
